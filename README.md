# SkyGlobal Airlines 🌍✈️

SkyGlobal Airlines is a fictional full-service international airline based in Dubai, United Arab Emirates. Founded in 2026, SkyGlobal aims to connect every corner of the world through a modern fleet, premium service, and a strong global network.

## Slogan
"Connecting Every Horizon"

## Main Hub
📍 Dubai, United Arab Emirates

## Fleet
- ATR 72-600
- Embraer E190
- Airbus A318-112
- Airbus A320-214
- Airbus A321-211
- Boeing 737-900ER
- Boeing 757-300 Winglets
- Airbus A340-300

## Brand Colors
- Deep Navy Blue (#0A2342)
- Gold (#D4AF37)
- Pearl White (#F8F6F0)

## Livery
SkyGlobal aircraft feature a clean pearl-white fuselage, elegant gold wave accents, and a navy-blue tail displaying the airline's globe logo, representing worldwide connectivity and premium travel.

## Vision
To become one of the world's leading airlines by connecting people, cultures, and destinations across every continent.

## Current Status
🚧 Growing Fleet
🌎 Expanding Route Network
🎨 Developing Special Liveries
✈️ Building a Global Aviation Brand
